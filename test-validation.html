🇰🇷 한글 맞춤법 검사기 테스트

테스트 방법:

  1. 아래 텍스트 박스에 "오랫만에 연락드립니다" 입력
  2. 1.5초 후 "오랫만에"에 빨간 wavy 밑줄 확인
  3. 오류 텍스트에 마우스 hover → 툴팁 표시 확인
  4. 수정 버튼 클릭 → 텍스트가 "오랜만에"로 교체되는지 확인
  5. 무시 버튼 클릭 → 빨간 줄이 사라지는지 확인

테스트 영역 1: Textarea

테스트 영역 2: Input Field

테스트 영역 3: ContentEditable

여기에 한글을 입력하세요.

예상 결과:

  • ✅ 수정 버튼: 텍스트가 제안된 내용으로 교체
  • ✅ 무시 버튼: 빨간 줄 사라지고 다시 API 호출할 때까지 표시 안됨
  • ✅ 툴팁 hover: 마우스 벗어날 때 2초 후 사라짐
